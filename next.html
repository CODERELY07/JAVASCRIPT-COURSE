ForEach

forEach: forEach is a method used to iterate over elements of an array in JavaScript. It executes a provided function once for each array element. The function receives three arguments: the current element, the index of the current element, and the array itself. forEach does not return a new array but is often used for its side effects, such as updating elements or performing actions for each element in the array.

Example:

Input:

todo.forEach(function(todos){
document.getElementById('forEach'). innerHTML += todos.text + "<br>"; })

Map

map is a method used to create a new array by applying a function to each element of an existing array. It returns a new array with the results of calling the provided function on every element in the array. The function provided to map can perform transformations on elements or extract specific data from them.

Example:

Input:

const todoText = todo.map(function(todos){
return todos.text;
});
document.getElementById('map').innerHTML += todoText;

Output:

Filter

filter is a method used to create a new array with all elements that pass a test specified by a provided function. It iterates over each element in the array and includes the element in the new array if the provided function returns true. filter does not modify the original array; it returns a new array containing only the elements that meet the specified condition.

Example:

const todoCompleted = todo.filter(function(todos){
return todos.isComplete === true;
}).map(function(todos){
return todos.text;
});
document.getElementById('filter').innerHTML += todoCompleted.join('
');
