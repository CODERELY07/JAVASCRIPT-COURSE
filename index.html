We can make a string into an array using the split() method

let arr = s.split(', ');
console.log(arr);

Object Destructuring

Object destructuring is a feature in JavaScript that allows you to extract multiple properties from an object and assign them to variables in a more concise way.

Example:

Input:

//object destructuring
const student = {
firstName: 'Mark',
lastName: 'Perez',
age:20,
hobbies: ['Music', 'Instrument', 'Coding'],
address: {
street: 'Sanlay Street',
city: 'Nabua',
}
}
const {firstName, hobbies, lastName} = student;

document.getElementById('object').innerHTML += `
Name: ${firstName} ${lastName}
Hobby: ${hobbies[0]}
`;

Array Of Object

Arrays of objects are commonly used in programming for organizing and managing structured data. For example, you might use an array of objects to represent a list of plans of users, where each student object contains properties like id, text, and isCompleted. This allows you to work with the data in a cohesive and organized manner, performing operations such as adding, removing, or modifying individual user records as needed.

Example:

Input:

//Array of Object
const plan = [
{
id: 1,
text: 'Be Creative',
isComplete: true
},
{
id: 2,
text: 'Be Disciplined',
isComplete: true
},
{
id: 3,
text: 'Be Consistent',
isComplete: true
}
];
plan.forEach(function(arr) {
document.getElementById('array').innerHTML += `
\tId: ${arr.id}
\tText: ${arr.text}
\tisComplete: ${arr.isComplete}

`;
});
